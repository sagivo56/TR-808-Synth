# יומן קלוריות (Calorie Tracker PWA)

אפליקציית ווב מסוג **PWA** (ניתנת להתקנה כאייקון על מסך הבית) לניהול יומן קלוריות
יומי, בעברית ו-RTL מלא. הערכת הקלוריות מתבצעת בצד שרת דרך **Anthropic Messages API**
(מפתח ה-API נשמר בצד שרת בלבד).

## תכונות

- **הגדרה ראשונית**: גובה, משקל, גיל, מין, רמת פעילות (5 רמות) ומטרה, עם חישוב יעד
  קלורי יומי לפי נוסחת **Mifflin-St Jeor** (TDEE = BMR × מקדם פעילות + התאמת מטרה,
  מעוגל ל-10 הקרוב, מינימום 1000).
- פתיחה מחדש של ההגדרות בכל רגע לחישוב יעד מחדש, וגם **עריכה ידנית מהירה** של היעד.
- **מד VU** עליון (28 מקטעים) שמתמלא לפי אחוז מהיעד: ירוק עד 85%, כתום 85%–100%,
  אדום מעל 100%. מתחתיו סה"כ קלוריות, "נשארו / חריגה", וסיכום חלבון/פחמימות/שומן.
- רשימת ארוחות היום עם שעה, קלוריות וכפתור מחיקה.
- **צ'אט דיווח על אוכל**: טקסט חופשי ו/או תמונה (עם הקטנה ודחיסה בצד לקוח —
  מקס' 1024px, JPEG ‎~0.82). המודל יכול לשאול שאלת הבהרה אחת, והשיחה נשמרת כ-thread
  (כולל התמונה המקורית) עד לתוצאה סופית.
- **שמירה מתמשכת בצד שרת** ב-SQLite: הגדרות פר משתמש וארוחות פר יום.
- **PWA אמיתי**: `manifest.webmanifest` תקין, אייקונים, `display: standalone`.

## דרישות מקדימות

- Node.js 18+ (מומלץ 20+)
- מפתח API של Anthropic — https://console.anthropic.com/settings/keys

## הרצה

```bash
# 1. התקנת תלויות
npm install

# 2. הגדרת משתני סביבה
cp .env.example .env.local
# ערכו את .env.local והכניסו את ANTHROPIC_API_KEY

# 3. (אופציונלי) יצירת אייקונים מחדש
npm run gen-icons

# 4. הרצה במצב פיתוח
npm run dev
# פתחו http://localhost:3000
```

לבנייה והרצת production:

```bash
npm run build
npm start
```

## משתני סביבה

| משתנה | חובה | ברירת מחדל | תיאור |
|-------|------|------------|-------|
| `ANTHROPIC_API_KEY` | ✅ | — | מפתח ה-API של Anthropic. **צד שרת בלבד**, לעולם לא נחשף בצד לקוח. |
| `ANTHROPIC_MODEL` | ❌ | `claude-opus-4-8` | מודל Claude להערכה (חייב לתמוך בקלט תמונות). |
| `DATABASE_PATH` | ❌ | `./data/app.db` | נתיב קובץ ה-SQLite. |

## התקנה כאפליקציה (מסך הבית)

1. פתחו את האפליקציה בדפדפן (Chrome/Edge/Safari).
2. בנייד: תפריט הדפדפן → **הוספה למסך הבית** (Add to Home Screen).
3. במחשב: אייקון ההתקנה בשורת הכתובת.

ה-`start_url` וה-`display: standalone` שבמניפסט גורמים לכך שההפעלה ממסך הבית תפתח
ישירות את מסך האפליקציה, ללא מעטפת דפדפן.

> הערה: התקנה כ-PWA דורשת הקשר מאובטח (HTTPS או `localhost`).

## מבנה טכני

- **Next.js 14 (App Router)** + **TypeScript** + **Tailwind CSS**
- **better-sqlite3** לשמירה מתמשכת בצד שרת
- **@anthropic-ai/sdk** לקריאה ל-Messages API מתוך `/api/estimate` (Node runtime)
- טיפול שגיאות מפורט: כשל בקריאת ה-API מציג את קוד ה-HTTP האמיתי, ושגיאת פענוח JSON
  מציגה את הסיבה ואת פלט המודל.

### נקודות קצה (API)

| Endpoint | פעולה |
|----------|-------|
| `GET/POST/PATCH /api/settings` | קריאה/שמירה/עדכון יעד ידני של ההגדרות |
| `GET/POST/DELETE /api/meals` | ארוחות היום: קריאה/הוספה/מחיקה |
| `POST /api/estimate` | הערכת קלוריות דרך Anthropic (מקבל thread, מחזיר question/result) |

זיהוי המשתמש נעשה דרך cookie (`uid`) שנוצר אוטומטית בפנייה הראשונה.
